# Namiwork

纳米Work 产品案例展示页（对外宣传用）。

## 页面内容

`index.html` 是一个零依赖的纯静态单页，包含：

- 首屏主视觉：核心 Slogan「一人成军，百事可托」+ 产品三阶段
- 三类核心人群（中小企业主 / OPC创业者 / 职场精英）Tab 切换展示
- 「动态网页交付」流程动效（生成 → 验收 → 分享 → 更新）
- 核心产品机制（专家智能体体系、专家教练、云端常驻、生态接入、企业级管理）
- 真实用户案例卡片墙（支持按人群筛选，案例内容均出自 `docs/纳米Work产品背景.md`，已脱敏）
- 纳米卫士三道防线安全背书
- 产品版本（个人版 / 企业版 / Box）
- 页面底部留资表单（试用意向收集钩子）

## 留资表单配置（部署前必做）

页面底部的留资表单需要配置数据接收方式。打开 `index.html`，搜索 `LEAD_CONFIG`，二选一：

- **`endpoint`**：填入表单收集服务的提交地址（如 Formspree、金数据等的 API 地址），表单数据将以 JSON POST 提交；
- **`email`**：不配置 endpoint 时的兜底方式——访客提交时会打开其邮件客户端，向该邮箱发送留资信息。**默认值是占位符，务必替换为真实邮箱后再对外发布。**

所有事实类内容以 `docs/纳米Work产品背景.md` 为唯一权威来源。

## 本地预览

无需构建工具，直接用浏览器打开 `index.html` 即可；或起一个本地静态服务：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 部署

纯静态页面，可直接托管：

- **GitHub Pages**：仓库 Settings → Pages → 选择分支根目录即可
- **Vercel / Netlify**：导入仓库，无需任何构建命令，输出目录为仓库根目录
