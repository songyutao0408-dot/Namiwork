# 轻提醒 · 到点就叫你

一个极简的日程提醒小工具，零依赖纯静态单页（`reminder/index.html`），对着浏览器直接打开就能用。

## 功能

- **输入文字**：写下要提醒的事，最长 200 字，`⌘/Ctrl + Enter` 快速添加。
- **选时间**：一排快捷时间（立即 / 5·15·30 分钟后 / 1·3 小时后 / 今晚 20:00 / 明早 09:00），也可点「指定时间」精确到分钟。
- **到点即时提醒**，三重保障：
  1. **系统通知**：授权后即使切走页面、锁屏也能收到（浏览器通知需页面在后台保持运行）。
  2. **页面弹窗**：铃铛动画 + 提醒内容，支持「知道了」或「5 分钟后再提醒」。
  3. **提示音 + 震动**：内置 WebAudio 蜂鸣（无需音频文件），手机端配合震动。
- **本地保存**：所有提醒存在浏览器 `localStorage`，刷新或重开页面自动恢复未到期的提醒，**不上传任何服务器**。
- **完成 / 删除**：点圆圈标记完成，点 × 删除；已完成的会归到下方列表。
- **深色模式**：跟随系统自动切换。

## 本地预览

无需构建，直接用浏览器打开 `reminder/index.html`；或起本地服务：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000/reminder/
```

## 使用小贴士

- 首次添加提醒时会请求「系统通知」权限，点允许后提醒更可靠。
- 浏览器对后台标签页的定时器有节流，**页面完全关闭后系统通知不保证触发**；如需锁屏/关页也能响，建议把页面「添加到主屏幕」以 PWA 方式常驻，或保持标签页开着。
- 数据只在当前浏览器、当前设备，换设备或清缓存不会同步。

## 技术

纯 HTML/CSS/JS 单文件，无框架无依赖。定时用 `setTimeout`（超长间隔自动分片重挂），提示音用 `AudioContext` 实时合成，通知用 `Notification API`，存储用 `localStorage`。
