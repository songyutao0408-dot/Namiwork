# 纳米Work 产品背景

> 本文档用于向 AI（如 Claude Code）提供纳米Work的产品背景知识，作为开发相关工具/网站时的上下文参考。内容仅涉及纳米Work自身的产品定位、机制与案例，不包含竞品分析。

---

## 一、产品是什么

**纳米Work**（360出品）是一款定位为"新一代 AI 工作平台"的产品，前身是 2026 年 3 月发布的"360 安全龙虾"，2026 年 6 月 24 日在 ISC 大会上正式升级更名为"纳米Work"。

**产品口号：一人成军，百事可托**

- **"一人成军"**：让一个人也能调动研究、销售、设计、开发、数据、运营等专业能力
- **"百事可托"**：AI 不只是陪聊，而是能接住任务、持续工作、交付成果

**产品能力进化的三个阶段**（360官方叙事）：
1. "装得上" —— 解决部署门槛问题
2. "用得好" —— 解决复杂工作流交付问题
3. "落得下" —— Box 硬件解决企业本地化部署

**核心定位一句话**：不是一个 Agent，而是一支帮你**谋**、帮你**干**、帮你**盯**的 AI 专家团队。

## 二、服务的三类核心人群

纳米Work不做"大而全的 Agent 货架"，而是聚焦三类高价值用户，围绕"分析—决策—交付"打通完整工作链：

### 1. 中小企业主 / 经营型创业者（用户占比约24%）
**用户心智**：已有生意，要从 1 做到 100；人少、预算有限，但获客、经营、营销一样都不能少。

- **找客户**：商机挖掘、客户画像、招投标监控、招商方案
- **看经营**：销售与利润分析、库存诊断、竞品与行业情报、经营决策
- **做增长**：营销策划、品牌物料、短视频、活动方案
- **省外包**：官网与小程序、视觉设计、宣传视频、数据工具

标志性价值：**一个人顶一支经营增长团队**

### 2. OPC / 一人公司创业者（用户占比约20%）
**用户心智**：尚在创业，从 0 到 1；没有完整团队，也要跑通创业闭环。

- **找方向**：赛道研判、需求调研、市场机会、政策信息
- **做产品**：网站、App、MVP、数据看板和轻量工具
- **找资源**：商业模式、BP、融资材料、园区与产业资源
- **跑自动化**：Skill开发、工作流搭建、数据处理、自动运营

标志性价值：**一个人完成从机会发现到产品验证**

### 3. 职场精英（用户占比约56%，占比最高）
**用户心智**：不是简单少干活，而是更快完成 KPI，持续交付专业成果。

- **做分析**：数据与财务分析、竞品情报、行业研究、经营洞察
- **出方案**：市场方案、销售方案、经营方案、项目报告
- **做汇报**：PPT、周报、述职、复盘、管理层汇报
- **推落地**：会议决策、任务拆解、项目跟进、风险提醒

标志性价值：**一个人具备咨询、分析、表达和推进能力**

## 三、核心产品机制

纳米Work的产品架构由几个核心模块组成：

### 1. 专家智能体体系
- 预置数百个"官方专家"智能体（专家市场），且**全部开放提示词，可二次训练**
- 用户可通过"问答式对话"孵化**自建专家**：把行业知识、判断标准、沟通风格、禁忌行为等写进专属专家配置，"会说话就能造出只懂你的专属专家"
- **专家教练**机制：基于最佳实践帮用户写提示词 → 全网搜索匹配最佳"技能"包 → 帮助设置定时任务 → 上传私有知识库 → 交付一个可持续进化的智能体
- 支持"专家团"（多智能体编组协作）完成复杂任务

### 2. 动态网页/轻应用交付能力
这是纳米Work区别于"只给答案/文件"的关键卖点：
- **可生成**：用户说清需求，纳米Work可生成动态网页、数据看板、交互报告、产品原型、轻应用
- **可验收**：生成不是结束，纳米Work会实际打开成果，检查页面展示、按钮点击、页面跳转、手机端适配，发现问题会继续修改，验收完成后再交付
- **可分享**：成果可生成 URL 或二维码，直接发给客户、团队、投资人或真实用户，对方无需安装软件，点开链接即可用
- **可更新**：需求或数据变化时，可持续通过对话迭代，交付的是可持续使用的在线成果，而非一次性静态文件

一句话总结："过去 AI 给你一段回答，后来能给你一份文件；现在纳米Work可以直接给你一个网址。"

### 3. 云端持续运行
- 支持 **7×24 小时云端常驻**工作，用户下班后 AI 继续干
- 支持**定时任务**，可自动化执行周期性工作（如每日数据播报、日报生成）

### 4. 中立开放的生态接入
- **不绑定单一模型**：接入国内外顶尖大模型，按任务动态调度
- **不绑定单一办公生态**：飞书、钉钉、企业微信均可对接，用什么接什么
- **系统打通**：支持与企业 ERP / CRM / OA / 自研系统对接（部分场景可"零改造接入"）

### 5. 企业级管理能力
- **透明组织树**：谁养了什么专家、沉淀了什么技能，一目了然
- **企业算力账本**：算力使用统一分配、用量可见、产出可查
- **技能一键蒸馏/发布**：一次做对，全员可复用

## 四、安全机制 —— 纳米卫士

360 的品牌基因是"安全"（深耕安全领域二十年），纳米卫士是贯穿产品的安全能力，主打**"为 AI 时代重写的三道防线"**：

- **可控读取**：AI 读取本地文件需授权，写入只发生在云端隔离空间
- **全程可溯 · 权限分级**：每一步操作都有记录，出问题可随时回退；谁能用什么、做到什么程度，由管理员设定
- **数据不出域**：企业数据不外泄

明确承诺：
- ✓ 不越权改文件
- ✓ 不产生计划外支出
- ✓ 不泄露企业数据
- ✓ 数据不出域

定位表述：不只是防病毒，而是**防 AI 被恶意指令劫持、防擅自下载、防接触不该接触的数据**。

## 五、产品版本

- **个人版**：让一个人拥有过去只有一个团队才能提供的专业能力，成为"超级个体"
- **企业版**：在个人版基础上，把个人的好用法沉淀成 Skill，把经验变成组织资产，把智能体接入企业流程，让一群超级个体组成"超级组织"；覆盖市场、销售、产品、财务、人力、研发、客服、供应链、老板等9大核心岗位
- **纳米Work Box**（企业本地化硬件）：软硬一体、本地部署，内置100+大模型、5万+技能、数百专家智能体，每台随机附赠算力（Token），主打持续运营和本地数据安全，配套渠道分销与工程师陪跑服务

## 六、典型真实用户价值案例（供理解产品使用场景参考）

以下为纳米Work真实用户案例中提炼的代表性场景，体现产品在不同岗位/行业的实际应用形态：

- **客服/运营管理者**：将海外多站点评分监控、全网比价、晋升述职PPT等日常重复工作交给"自建专家"，AI 自动拉数、生成图片日报、比价表、演示页，管理者只负责下指令和纠错。
- **智能硬件业务负责人**：将广告数据盯盘设为定时播报（整点自动出进度表并预警）、把数字员工拉进工作群参与竞品对比与方案评审讨论。
- **客服运营负责人**：搭建自建专家自动生成呼叫中心 KPI 日报、智能外呼质检表、海外客服培训课件与分析报告，减少人工翻表工作量。
- **用户体验负责人**：新品上市期让专家自动读取飞书跟踪表、按统一口径统计新增反馈、批量打标生成结构化标签词典，实现"一人盯住整条新品反馈线"。
- **工程项目管理者**：上传项目概算 Excel 与招标文件，纳米Work自动解析测算监理费用是否合理，并从建设方视角逐条排查招标合同中的风险条款，当天出结论。
- **餐饮/酒店/工程咨询等中小企业**：训练出专属的品牌SOP专家、全栈开发+顾问团、研发+战略顾问团，实现"一个人指挥一支队伍"的经营效果，无需额外招聘。

这些案例共同体现的产品价值：**用户提供目标、资料和截止时间，纳米Work完成"读资料、拆任务、调专家、找信息、做成品、按意见反复改"的完整流程，最终交付 Word / PPT / Excel / 网页 / 应用 / 图片等可持续使用的成果。**

## 七、给开发者/AI 的建议使用方式

如果基于本背景资料开发纳米Work相关的宣传工具、案例展示网站、内部演示Demo等，建议重点突出以下几个差异化要素：

1. **"专家团"叙事**：不是单一Agent助手，而是可训练、可协同的专家团队
2. **"动态网页交付"能力**：这是区别于"只给文字/文件"类 AI 工具的核心亮点，适合做成可视化演示
3. **三类人群的场景化区分**：中小企业主（经营增长）、OPC创业者（创业成事）、职场精英（职场交付），文案和功能展示可按人群分层设计
4. **安全信任背书**：360安全基因 + 纳米卫士三道防线，适合作为企业级/toB场景的信任状
5. **真实案例驱动**：产品资料中沉淀了大量真实用户案例（数据来源于用户真实使用记录），可用作社会认同类素材，但涉及具体企业/个人信息时注意脱敏处理
